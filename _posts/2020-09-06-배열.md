* #### 정수형 배열

```java
// 정수형 배열 odd 생성
int[] odd = {1, 3, 5, 7, 9}; // odd의 크기는 5
// 배열 출력
System.out.println(odd[0]); // 1 출력
// 배열의 크기 출력
System.out.println(odd.length);
```


* #### String 형 배열

```java
// 배열 생성
String[] weeks = {"월", "화", "수", "목", "금", "토", "일"};
```
