### 조건문  
#### - if(조건) {  
//조건이 참일 때 실행  
} else {  
    //조건이 거짓일 때 실행  
}  
  
```JAVA
if(flag == false) {
			System.out.println("flag 는 " + flag + "입니다.");
		} else {
			System.out.println("flag 는" + flag + "입니다.");
		}
```
  
#### - 삼항연산자  
(조건) ? 참일 때 실행 : 거짓일 때 실행  

```JAVA
String resultStr = "";
		resultStr = (age >= 19) ? "성인인증 완료" : "미성년자입니다.";

```
