### 변수 
- int : 정수형(기본) 변수
- double, float : 실수형(기본) 변수
- boolean : 논리형(기본) 변수
- String : 문자열(참조) 변수
- char : 문자형 변수

```java
    int age = 32;
		double pi = 3.14;
		boolean flag = true;
		String name = "티모";
```
---
**※ 변수 선언시 주의사항**
- 변수는 숫자로 시작 불가
- _(언더스코어)와 $이외에 특수문자는 사용 불가
- 자바의 키워드는 변수명으로 사용 불가 (ex. for, char, public...)
- 변수명으로 a, b등은 사용하지 않는다.

#### <자주 쓰는 자료형 변수>
int, long, double, boolean, char, String, List, Map
