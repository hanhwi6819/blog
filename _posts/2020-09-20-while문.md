#### while문 : 조건동안 반복

```java
while(state<100) {
	++state;
	plusVariable += state;
}
```


#### while문 중단

```java
boolean flag = true;
int state = 0;
		
while(flag) {
	++state;
	System.out.println("미로입니다.");
		if (state == 10) {
				
			System.out.println("탈출했습니다.");
			// flag = false; : while 문을 멈추는 코드
			break; //while 문을 멈추는 코드
		}
}

```
▶ 조건을 거짓으로 바꾸거나 break 사용
